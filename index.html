Task Details

Prepare clue cards

Phase 1: Planning 🔒 Required
📅
WHEN
1-2 weeks before event

📝 Instructions

  1. Decide number of clues (10-15 works best)
  2. Create a mix of easy, medium, and hard clues
  3. Write location hints that aren't too obvious
  4. Add fun riddles or puzzles where possible
  5. Print on cardstock for durability
  6. Number each card for tracking

💡 Pro Tips

From Creator

"Laminate the clue cards if playing outdoors - learned this the hard way when it started drizzling! Also, take photos of where you place each clue, you'll thank yourself later."

— EventPro Studios

🔄 Flexibility

For kids (under 10), use picture-based clues instead of text. For corporate events, add company trivia questions as clues.
